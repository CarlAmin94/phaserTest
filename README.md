# Exemples d'applications réalisées avec PhaserJS

Ce git contient plusieurs applications *PhaserJS*. Ceci à pour but d'approcher ce framework
Le lancement d'un application est très simple : Il suffit de lancer le *<nom application>.html*
Chaque .html utilise le script *phaser.min.js* ainsi que le contenu du dossier *assets* pour les ressources (images)

# Liste des applications

## Zack Attack
On contrôle un personnage (Zack du jeu vidéo Final Fantasy 7). Le but est de ramasser les materias rouges. Le joueur peut se déplacer de gauche à droite et sauter.
